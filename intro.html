1  Introduction


R was designed specifically for statistical analysis and is extremely well suited to the tasks facing economists. Whether it’s econometrics, time series, advanced statistical analysis or predictive models, R can really do the job.

What’s more, it’s free, which makes it a cool language!

In this tutorial, you will learn how to:

  • Manipulate, transform, merge and synthesize data using the tidyverse package
  • Create simple loops and functions
  • Perform all kinds of econometric models and statistical analyses
  • Make beautiful visuals and a clean document to share

You can download the latest versions of R and R Studio. Then simply copy/paste the lines of code and run them on your computer.

Important

The techniques you’ll learn here are very similar to those used with Python, SQL, SAS, STATA, Julia etc… No matter how motivated you are to follow this tutorial, it’s not a waste of time! See the different codes here:

use data.csv, clear;
keep if age >= 18;
summarize salary;

using CSV
using DataFrames
data = CSV.read("data.csv")
filtered_data = data[data.age .>= 18]
grouped_data = data |> groupby(age) |> mean(salary)

This example reads a CSV file, filters the data to retain only those aged 18 or over, then groups the data by age and calculates the average salary for each age group. The syntax is very similar in all six languages, making it easy to transfer code between them. Rstudio can also incorporate Python, SQL, SAS, STATA or Julia code!

You may also notice that R and Python work with libraries (or packages). Here dplyr for R and pandas for Python. We’ll take a closer look at the use of these packages in section 1.2.

1.1.1 R studio

RStudio is an integrated development environment (IDE) for the R programming language. It provides a graphical user interface (GUI) for writing, editing, and running R code. It also includes a number of features that make it easier to work with data, such as a code editor, a debugger, and a console. RStudio is a popular IDE for R, and it is used by a wide range of users, from students to researchers to data scientists.

Here are some of the key features of RStudio:

  • A code editor with syntax highlighting and auto-completion
  • A debugger that allows you to step through your code line-by-line
  • A console where you can run R commands
  • A variety of tools for working with data, such as a data viewer and a data frame editor
  • A built-in help system

RStudio is a powerful tool that can help you be more productive when working with R. If you are new to R, we highly recommend using RStudio. It will make your learning experience much easier.

Here are some additional details about RStudio:

  • It is available for Windows, macOS, and Linux.
  • It is free to use for personal use.
  • There is a paid version of RStudio that includes additional features, such as a cloud-based IDE and a subscription to the RStudio Pro services.

The functions of these different sections are as follows:

  • A: Source file editing window. Here you can edit your script file, select parts and execute them with Ctrl-Enter (Cmd-??? on Mac). Data table tabs are also displayed here.

  • B: Environment window. This displays the list of objects you have created. These objects can be simple variables, tables, functions, graphical objects, etc.

  • C: Console. Here you can write command lines directly. Press Enter to execute a line you’ve just written. This is also where any error messages are displayed.

  • D: Multi-tabbed window. Files is used to navigate through your computer’s file system, Plots displays graphics, Packages displays available and loaded packages (when checked). Help provides help on functions.

1.1.2 Packages

A package in R is a collection of R code, data, and documentation that can be installed and used as a single unit. Packages are a great way to share code with others, and they can be used to extend the functionality of R.

R packages are stored in the R Package Archive (CRAN), which is a central repository of R packages. Packages can be installed from CRAN using the install.packages() command.

Once a package has been installed, it can be loaded into the R environment using the library() command. Once a package has been loaded, its functions and data can be accessed.

R packages are a powerful tool that can be used to improve the productivity and efficiency of statistical analyses.

  • dplyr: This package is used for data manipulation. It provides a consistent and intuitive way to filter, select, mutate, and summarize data frames.

  • tidyr: This package is used for tidying data. It provides a set of functions to help you transform your data into a format that is easy to work with.

  • ggplot2: This package is used for creating beautiful and informative graphics.

  • readr: This package is used for reading data into R. It provides a consistent and efficient way to read data from a variety of file formats.

  • stringr: This package is used for working with strings. It provides a set of functions to help you manipulate, search, and parse strings.

  • lubridate: This package is used for working with dates and times. It provides a set of functions to help you parse, manipulate, and format dates and times.

  • purrr: This package is used for functional programming. It provides a set of functions that make it easy to write concise and elegant code.

  • forcats: This package is used for working with factors. It provides a set of functions to help you manipulate, summarize, and visualize factors.

These are just a few of the many R packages that are available. There are packages for almost everything you can think of, from machine learning to statistical modeling to visualization.

1.1.3 An object-oriented programming language

As Python, R is an object-oriented programming language, that is to say a programming paradigm based on the concept of objects, which can contain data and code. Objects in R can be vectors, matrices, arrays, data frames, tables and list.

Every time you see this symbol <-, it means that R is creating an object that it will keep in memory. So, if I do this A <- 6, A will have become an object, and logically sqrt(A) will be equal to \(\sqrt{6}\).
