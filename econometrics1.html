7  Econometrics

7.1 The Wooldridge package

What better tool than R for econometrics? It’s in its very DNA: R was originally a statistical software specifically designed for all kinds of econometric analysis. In this first chapter devoted to econometrics, we’re going to use a very useful package based on one of the greatest references in econometrics: Introductory Econometrics: a modern approach by Wooldridge (2015).

Based on this book, some R users have done us a great service and published an extremely useful package that we will use in these two sessions: package(wooldridge). You will find all the explanations on the different databases included, variable names, nature, regression example etc… here: Wooldridge package

This package contains 115 databases used in the book. You can then easily reproduce the book’s regressions using the same data as Wooldridge. Very practical for learning econometrics! A simple command displays them all:

7.2 Single regression

We want to estimate the parameter of the following regression:

\[ lwage_i = \beta_0 + \beta_1 educ_i + \epsilon_i \] With:

  • lwage: the log wage
  • educ: the level of education in years
Important
  • \(Y = \beta_0+\beta_1 X\): \(X\) increases by one unit, \(Y\) increases by \(\beta_1\) units.
  • \(\log(Y) = \beta_0+\beta_1 X\): \(X\) increases by one unit, \(Y\) increases by \(\beta_1 \%\) units.
  • \(\log(Y) = \beta_0+\beta_1 \log(X)\): \(X\) increases by \(1 \%\) unit, \(Y\) increases by \(\beta_1 \%\) units.

The easiest way to do so is:

log_wage_model <- lm(lwage ~ educ, data = wage1)
summary(log_wage_model)

7.3 Multiple regression

7.3.1 Wage

What if we want a multiple regression? Nothing easier: just add the name of the variable you need to. For instance, for

\[ lwage_i = \beta_0 + \beta_1 educ_i + \beta_2 exper + \beta_3 tenure + \epsilon_i \] With:

  • lwage: the log wage
  • educ: the level of education in years
  • exper: the number of year’s experience in working
  • tenure: the number of years in the firm
log_wage_model2 <- lm(lwage ~ educ + exper + tenure, data = wage1)
stargazer(log_wage_model2, type = "text", single.row = TRUE, header = FALSE)

7.3.2 Effects of Pollution on Housing Prices

We will now study the effect of pollution on the price of a house using two models. In the first, we want to run the following regression:

\[ \log(price) = \beta_0 + \beta_1\log(nox) + \beta_2\log(dist) +\beta_3rooms + \beta_4stratio + \epsilon \]

Where

  • price: the price of the house
  • nox: Nitrous Oxide concentration; parts per million.
  • dist: weighted distance of the community to 5 employment centers.
  • rooms: the number of rooms
  • stratio: average student-teacher ratio of schools in the community.

The second model will perform:

\[ \log(price) = \beta_0 + \beta_1\log(nox) + \beta_2\log(dist) +\beta_3rooms + \beta_4 rooms^2 + \beta_5stratio + \epsilon \]

price <- lm(lprice ~ lnox + log(dist) + rooms + stratio, data = hprice2)
price2 <- lm(lprice ~ lnox + log(dist) + rooms+I(rooms^2) + stratio, data = hprice2)
stargazer(price, price2, type = "text", single.row = TRUE, header = FALSE)

In the first model (i.e. without taking into account the \(rooms^2\) effect), we can see that adding an additional part increases the price by around 25.5%. This is very different from model 2, where the effect is 25.5% for a value of \(rooms\)=6.45, but changes rapidly when \(rooms\) increases or decreases.

This is because, from a certain point onwards, the effect becomes positive and the quadratic shape means that the semi-elasticity of \(price\) with respect to \(rooms\) increases as \(rooms\) increases.

7.4 Learning by doing

Another very useful tool to learn econometrics is provided by the Wooldridge Vignette made by Justin M Shea. It aims to present 18 econometrics Technics for many different situations, including time series.

For the 3 studies below, run the code and try to understand which techniques are used.

# 1: the ivreg function
wage_educ_model <- lm(lwage ~ educ, data = mroz)
fatheduc_model <- lm(educ ~ fatheduc, data = mroz, subset = (inlf==1))
library("AER") # g the ivreg function in the AER (Applied Econometrics with R) package.
wage_educ_IV <- ivreg(lwage ~ educ | fatheduc, data = mroz)
stargazer(wage_educ_model, fatheduc_model, wage_educ_IV, type = "text", single.row = TRUE,
header = FALSE)

# 2: Wages and productivity
wage_time <- lm(lhrwage ~ loutphr + t, data = earns)
wage_diff <- lm(diff(lhrwage) ~ diff(loutphr), data = earns)
stargazer(wage_time, wage_diff, type = "text", single.row = TRUE, header = FALSE)

# 3: Number of arrest
formula <- (narr86 ~ pcnv + avgsen + tottime + ptime86 + qemp86 + inc86 + black +
hispan + born60)
econ_crime_model <- lm(formula, data = crime1)
econ_crim_poisson <- glm(formula, data = crime1, family = poisson)
stargazer(econ_crime_model, econ_crim_poisson, type = "text", single.row = TRUE, header = FALSE)
