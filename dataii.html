4.1.1 Difficulty

  1. The first obvious difficulty you’re going to encounter is that there are 4 different databases, so you’re going to have to merge them. But on what criteria?

  2. There’s a lot of data missing in all directions. We’ll have to make decisions about them: delete them, replace them with averages, medians, forecasts? If you want to display a function without taking into account the NAs, just add na.rm = TRUE in your code.

4.1.2 Roadmap

We’re going to work in stages! And before moving on to merging the data, it’s a good idea to clean up the databases and check that they’re in the right order, one by one. We’ll then merge them in a second stage, retaining all the data. We’ll work in 4 stages:

  • Step 1: Clean and make tidy the four bases separately

  • Step 2: Merge the data

  • Step 3: Prepare a dataase for the three tasks

  • Step 4: Render graphs and tables

4.2 Create a unique Database

4.2.1 Step 1

Start by creating 4 new objects, df1, df2, df3 and df4 to store the input data.

See solution 
df1 <- GDP
df2 <- Inflation
df3 <- unempl
df4 <- Country

Then work on each base separately:

What should we do?

We just have to make it tidy!

Tip

See data I and reproduce the same gather() operation. Be careful with the number of columns!

Solution 
df1 <-gather(df1,year,gdp,3:65)
df2 <-gather(df2,year,inflation,3:64)
df3 <-gather(df3,year,unemployment,3:65)

4.2.2 Step 2

What should we do?

The aim is to merge the four bases into a single one. The first step is to merge df1 with df2 and then with df3 and df4 (the only static database). Then keep only the column displayed in the final table. Call the final table df.

Tip

Now it’s crucial to merge the data with a common key. In the three dynamic databases, i.e. those containing information by year, we can no longer use only the country name as a key. All we need to do is add the year to the by = c("") argument of the join function. Be careful with column names!

Solution 
df <- full_join(df1,df2,by=c("country","year"))

colnames(df3)[1] <- "country"
df <- full_join(df,df3,by=c("country","year"))

colnames(df4)[2] <- "country"
df <- full_join(df,df4,by=c("country"))

df <- df[,-c(2:2,5:5,7:7,9:11)]

write.csv(df, "DataEco.csv",row.names = FALSE)

4.3 France Phillips Curve

What should we do?

We need to create a graph with the unemployment rate on the x-axis and the inflation rate on the y-axis. The data must be filtered beforehand, so as to keep only France for the years in which there are no missing data.

Tip

Use de filter() function in tidyverse to select only France data. Then use again filter() to keep only available data on inflation and unemployment. The command !is.na() will help you to erase missing data. For the graph, you can try the ggplot package with:

ggplot(France, aes(x = , y = )) +
  geom_point(aes(color = factor())) +
  stat_smooth(method = "lm",
              col = "#C42126", se = FALSE, size = 1
  )

The method="lm" command will show the linear regression.

Solution 
France <- filter(DataEco,country=="France")

France <- filter(France, !is.na(inflation))
France <- filter(France, !is.na(unemployment))

ggplot(France, aes(x = unemployment, y = inflation)) +
  geom_point(aes(color = factor(year))) +
  stat_smooth(method = "lm",
              col = "#C42126", se = FALSE, size = 1
  )

4.4 Top 15 Inflation Mean

What should we do?

The aim is to create a ranking of data by country, based on the average inflation rate over all the years observed, and then to retain only the top 15 values.

Tip

You’ll have to use the %>% pipe function! Take another look at the “Learning by Doing” section of Data I, where the groupe_by(), summarize(), top_n() and arrange() functions are used.

Solution 
inf_mean <- DataEco %>% group_by(country) %>% summarise(mean_inf = mean(inflation,na.rm = TRUE))
inf_mean <- inf_mean %>% top_n(15, mean_inf) %>% arrange(desc(mean_inf))

4.5 Eurozone GDP

What should we do?

The aim is to create a dynamic graph of the sum of Euro zone countries GDP.

Tip

You’ll have to use the dygraph package. First, filter the data by keeping only countries that have euro as a currency, and keep only years after 2000. Then, create a base with 2 columns, “year” and “GDP”, which is the sum of all countries gdp and call the base gdp_eur. finally, run dygraph(gdp_eur).

Solution 
gdp_eur <- DataEco %>% filter(Currency=="EUR", year > 2000)
gdp_eur <- gdp_eur %>% group_by(year) %>% summarise(GDP = sum(gdp,na.rm = TRUE))

gdp_eur$GDP <- gdp_eur$GDP/1000000000

library(dygraphs)

dygraph(gdp_eur)
